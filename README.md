# Генератор макетов для закладок
Привет! Это скрипт, который помогает печатать закладки с двух сторон, даже если у твоего принтера только односторонняя печать (например, как у Canon G1411). Он сам расставляет картинки на листе А4 и главное — делает "зеркальный" макет для обратной стороны, чтобы всё ровненько совпало при печати.

Автор: @pingalinux

## Что он умеет?
*   Берёт картинки из папки и аккуратно раскладывает их на листе.
*   Понимает обычные картинки (`.png`, `.jpg`) и даже файлы `.studio3`.
*   Делает правильный оборот для двусторонней печати: картинки для "спины" зеркалятся и сдвигаются как надо.
*   Сам добавляет отступы, чтобы можно было удобно резать.

## Что нужно?
Нужен Python и всего одна библиотека `Pillow` (она работает с картинками).

Ставится так:

```bash
pip install pillow
```

## Как тут всё устроено?
```text
.
├── bookmark_layout_generator.py  # Главный файл, запускай его
├── config.py                     # Все настройки (DPI, поля, пути)
├── utils.py                      # Вспомогательные функции для картинок
├── layout_engine.py              # Логика расстановки закладок на листе
├── input/                        # Сюда кидай исходные картинки
└── output/                       # А здесь появятся готовые макеты
```

## Как пользоваться?
1.  Закинь картинки своих закладок в папку `input`.
2.  Запусти скрипт:

    ```bash
    python bookmark_layout_generator.py
    ```

3.  Он немного подумает и положит готовые листы в папку `output`.
4.  А ещё он прямо в консоли напишет, как это всё печатать, чтобы не запутаться.

## Самое главное: Как печатать?
Чтобы лицо и оборот совпали, делай так (для одностороннего принтера):
1.  Сначала печатай **ВСЕ файлы `_front`** (это лицо).
2.  Возьми всю стопку напечатанного и **переверни её СЛЕВА НАПРАВО** (как будто листаешь страницу книги).
3.  Засунь стопку обратно в принтер.
4.  Теперь печатай **ВСЕ файлы `_back`**, но **В ОБРАТНОМ ПОРЯДКЕ** (начни с последнего файла).

Если всё сделаешь по инструкции, закладки получатся идеальными!
